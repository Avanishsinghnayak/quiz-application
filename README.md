# Quiz App (MCQ)

## How to run
- Open `index.html` in your browser (double-click it).

## Features (as required)
- Start quiz button
- Displays one question at a time
- 6 MCQ questions (4 options each)
- Single selection per question + highlighted selected option
- Next button navigation (disabled until you pick an option)
- Score calculated and shown at the end
